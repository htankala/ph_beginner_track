# Pearl Hacks Bakery Adventure

Welcome to the Pearl Hacks Beginner Python Workshop!

In this workshop, you will build a text-based bakery adventure game using:
- Variables
- Functions
- Conditionals
- Loops
- Dictionaries
- Basic input validation

---

## Setup Instructions

## Install VS Code (If You Don't Have It)

1. Go to: https://code.visualstudio.com/
2. Click **Download for your operating system**
3. Install VS Code like a normal application
4. Open VS Code after installation

### Install the Python Extension (Important!)

1. Open VS Code
2. Click the **Extensions icon** (left sidebar, looks like squares)
3. Search for **Python**
4. Install the extension by Microsoft

---

## Step 2: Install Python

1. Go to: https://www.python.org/downloads/
2. Download Python 3 (3.10+ recommended)
3. During installation:
   - Check "Add Python to PATH"
4. Verify installation by running in Terminal:

## Step 3: Check Requirements and Clone Repository

1. Make sure you have Python 3 installed: `python --version`
3. Clone this repository: `git clone <repo-link>`
4. Navigate into the starter folder: `cd pearl-hacks-bakery-beginner/starter`
5. Run the program: `python main.py`

---

## Your Goal

Complete the TODOs inside the `starter/` folder to:

- Manage bakery inventory
- Help customers
- Bake cupcakes
- Earn money
- Restock ingredients
